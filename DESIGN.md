# Paper Reviewer App Design

## User story

As a reviewer, I want to read a paper, highlight sections, add comments, and keep notes organized so I can write better reviews.

## Core features

- Manuscript reader with text selection
- Inline highlights and colors
- Comment panel linked to selected text
- Reviewer tags: `Strength`, `Weakness`, `Question`, `Suggestion`
- Section-based review notes
- Export summary as markdown or copyable text

## UI layout

- Left pane: manuscript viewer
- Right pane: annotation/comment panel
- Top toolbar: highlight, tag, export
- Floating menu on selection for quick actions

## Future improvements

- PDF annotation support
- Save/load review sessions
- AI review suggestions
- Checklist for common review criteria
- Sync to cloud or Git for versioned review notes
